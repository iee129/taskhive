# 프로젝트 개요

## TaskHive란?

TaskHive는 팀이 프로젝트와 할 일을 체계적으로 관리할 수 있는 풀스택 웹 애플리케이션입니다. 벌집(Hive)처럼 각 구성원이 유기적으로 연결되어 협력하는 환경을 지향합니다.

## 핵심 가치

| 가치 | 설명 |
|------|------|
| **명확성** | 누가 무엇을 언제까지 해야 하는지 한눈에 파악 |
| **협업** | 팀원 간 태스크 할당과 진행 상황 공유 |
| **단순성** | 복잡한 설정 없이 즉시 사용 가능한 인터페이스 |

## 타겟 사용자

- 소규모 팀(2~10명)의 프로젝트 관리자
- 개인 할 일 관리가 필요한 개발자
- 스프린트 기반 작업 추적이 필요한 팀

## 주요 기능

- **회원 인증**: 이메일/비밀번호 기반 JWT + Refresh Token 이중 구조, Role 기반 접근 제어
- **태스크 관리**: 생성·수정·삭제·상태 변경(TODO → IN_PROGRESS → DONE), 우선순위·담당자·댓글
- **프로젝트 구성**: 태스크를 프로젝트 단위로 그룹화, 칸반 보드 뷰
- **AI 어시스턴트**: 자연어로 태스크 생성 ("다음 주까지 보고서 작성"), 일간 태스크 요약 다이제스트 (Ollama 기반)
